# MiseEnPlace

Family meal planning calendar with inventory management.

## Stack
- **Frontend**: Next.js + Tailwind CSS + Vercel
- **Backend**: Supabase (Auth + PostgreSQL)
- **Deploy**: Vercel (frontend)

## Concept
A shared family calendar for planning meals, assigning responsibilities, and managing home inventory.

## Features
- 📅 Shared calendar for meal planning
- 👥 User authentication and responsibility assignment
- 📦 Inventory management for household ingredients
- 🛒 Shopping list with inventory integration

## Getting Started

1. Clone the repository
2. Copy `.env.example` to `.env.local` and fill in your Supabase credentials
3. Run `npm install`
4. Run `npm run dev`

## Database Setup

Run the SQL in `supabase-schema.sql` in your Supabase SQL Editor to create the required tables.
